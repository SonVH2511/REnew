### REnew

An IDA plugin that automate reload your current binary, don't need to close file and reopen ida...Maybe more convenient :))
### Requirement

Work on 9.0+ version

### Usage

Just throw file `reload_binary_plugin.py` to plugins folder `%APPDATA%\Hex-Rays\IDA Pro\plugins\`.

Shortcut: ctrl + shift + R.
